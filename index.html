fLaShY.JS

flashy.js is a script that helps you make cool animated borders, like on this page :) :) :)!!! It comes with two AWESOME built-in color sets, $flashy.rainbow and $flashy.grayscale! It is invoked with $flashy.MY_EYES(target, [options]);, where target is the victim element!:):)! flashy.js uses jQuery!!!!!!!

Example

H.T.M.L. snippet

<p id="lol">
  Proin vel ligula at risus adipiscing facilisis.
  Ut dolor metus, volutpat vel congue tempor, iaculis eu lectus.
</p>

JavaScript snippet

/*                 \/ element we are targeting
                        \/ delay in milliseconds
                             \/ border-width
                                 \/ color set   */
$flashy.MY_EYES('#lol', 100, 10, $flashy.rainbow);
